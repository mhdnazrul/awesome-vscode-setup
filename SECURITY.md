# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do **not** open a public issue.

Send a private report to: mhdnazrul511@gmail.com

You should receive a response within 48 hours. If not, follow up to ensure the message was received.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential impact

## Scope

This repository contains VS Code configuration files and scripts. Vulnerabilities typically relate to:

- Scripts that modify user files or directories
- Extensions with known security issues
- Secrets or credentials accidentally committed
